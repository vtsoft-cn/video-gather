# videoGather

#### 介绍
视频资源采集程序 - 集成前端页面，该项目采集的视频数据资源均来自互联网分享站点所提供的公开引用资源，未提供资源上传、存储服务

#### 软件架构
SpringBoot + thymeleaf + h2
